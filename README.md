# PRO Trainer

PRO Trainer is a Python bot to play the game [Pokemon Revolution Online](https://pokemonrevolution.net/home).

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install
```
